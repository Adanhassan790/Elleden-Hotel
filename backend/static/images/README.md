# Elleden Hotel - Required Images

Place all images in this folder (`backend/static/images/`).

## Required Images

### Homepage
| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| `hero-bg.jpg` | Main hero background image (hotel exterior/lobby) | 1920x1080 px |
| `hotel-lobby.jpg` | Hotel lobby/reception area | 800x600 px |
| `cta-bg.jpg` | Call-to-action section background | 1920x600 px |

### About Page
| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| `hotel-exterior.jpg` | Hotel building exterior | 800x600 px |

### Rooms
| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| `room-single.jpg` | Single room photo | 600x400 px |
| `room-double.jpg` | Double room photo | 600x400 px |
| `room-deluxe.jpg` | Deluxe room photo | 600x400 px |
| `room-suite.jpg` | Suite room photo | 600x400 px |

### Services
| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| `restaurant.jpg` | Restaurant interior | 800x600 px |
| `catering.jpg` | Catering service photo | 800x600 px |
| `conference.jpg` | Conference room photo | 800x600 px |

### Optional (Nice to Have)
| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| `logo.png` | Hotel logo (transparent background) | 200x80 px |
| `favicon.ico` | Browser tab icon | 32x32 px |

## Image Tips
- Use high-quality JPG images for photos (80-90% quality)
- Use PNG for logos and graphics with transparency
- Optimize images for web (compress to reduce file size)
- Keep aspect ratios consistent for similar image types

## After Adding Images
1. Save images with the exact filenames listed above
2. Restart the Django server (Ctrl+C, then `python manage.py runserver`)
3. Refresh your browser (Ctrl+F5 for hard refresh)
