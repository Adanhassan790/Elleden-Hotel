# Elleden Hotel (K) Limited - Website

A professional, responsive website for Elleden Hotel (K) Limited, a premier hospitality establishment located in Marsabit, Kenya.

## 📁 Project Structure

```
website/
├── index.html          # Home page
├── about.html          # About Us page
├── rooms.html          # Rooms & Accommodation page
├── restaurant.html     # Restaurant & Dining page
├── conference.html     # Conference Facilities page
├── catering.html       # Outside Catering Services page
├── contact.html        # Contact Us page
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   └── main.js         # JavaScript functionality
└── images/             # Image assets (add your images here)
```

## 🚀 Getting Started

1. **Add Images**: Place your hotel images in the `images/` folder with the following names:
   - `hero-bg.jpg` - Hero section background
   - `hotel-lobby.jpg` - About section image
   - `cta-bg.jpg` - Call-to-action background
   - `room-single.jpg` - Single room image
   - `room-double.jpg` - Double room image
   - `room-twin.jpg` - Twin room image
   - `room-family.jpg` - Family room image
   - `restaurant.jpg` - Restaurant image
   - `catering.jpg` - Catering services image

2. **Open in Browser**: Simply open `index.html` in your web browser to view the website.

## ✨ Features

- **Fully Responsive**: Works on all devices (desktop, tablet, mobile)
- **Modern Design**: Clean, professional look with smooth animations
- **7 Pages**: Complete website with all hotel services
- **Contact Form**: Interactive form with validation
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Smooth Scrolling**: Enhanced user experience
- **Animation Effects**: Scroll reveal and hover animations
- **SEO Ready**: Proper semantic HTML structure

## 📱 Pages

1. **Home** - Welcome page with services overview
2. **About** - Company information, mission, vision, values
3. **Rooms** - Accommodation details and room types
4. **Restaurant** - Dining services and menu preview
5. **Conference** - Conference facilities and packages
6. **Catering** - Outside catering services
7. **Contact** - Contact information and inquiry form

## 🎨 Customization

### Colors
Edit the CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #8B7355;
    --secondary-color: #D4AF37;
    /* ... */
}
```

### Fonts
The website uses:
- **Playfair Display** - Headings
- **Poppins** - Body text
