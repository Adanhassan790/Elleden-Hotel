================================================================================
STEPS TO ADD VARIABLES IN RAILWAY DASHBOARD
================================================================================

1. Go to https://railway.app
2. Select your Elleden Hotel project
3. Click on your app service (the main one)
4. Go to the "Variables" tab
5. Click "+ New Variable"
6. Enter the Key and Value from above
7. Click "+ New Variable" for each additional variable
8. After adding all variables, click "Save & Redeploy"

================================================================================
HOW TO FIND YOUR APP DOMAIN
================================================================================

After successful deployment:
1. Go to Railway Dashboard
2. Select your project
3. Look at the main service card
4. Your domain will be shown as: https://yourapp-abc123.railway.app
5. Note this domain and use it to replace "your-app-name" in:
   - ALLOWED_HOSTS
   - MPESA_CALLBACK_URL

================================================================================
DATABASE SETUP
================================================================================

1. In your Railway project, click "+ Add Service"
2. Select "Database" → "PostgreSQL"
3. Railway will automatically:
   - Create a PostgreSQL database
   - Set the DATABASE_URL environment variable
   - Run migrations via Procfile release command
